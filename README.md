# Akquise Wolf – Sprint 1

Moderne Telefon-Akquise-Software für Kunden-Leads und Netzwerk-Aufbau.

## Funktionen Sprint 1

- Dashboard im Akquise-Wolf-Look
- Telefon-Session mit 1-Klick-Anruf (`tel:`)
- Kunden-Leads und Netzwerk-Aufbau
- Manueller Lead über „Neuer Lead“
- Schnell-Lead über „Schnell-Lead“
- Excel/CSV Import mit Vorschau und Fehlerprüfung
- Excel/CSV Export mit Filtern
- Ampel-System und Wiedervorlagen-Grundlogik
- Lokaler Browser-Speicher
- Responsive für Handy, Tablet und Desktop
- Cloudflare Pages kompatibel

## Lokal starten

```bash
npm install
npm run dev
```

## Cloudflare Pages

Framework: Vite  
Build command: `npm run build`  
Output directory: `dist`

## Hinweis

Sprint 1 speichert Daten lokal im Browser. Sprint 2/3 ergänzt Cloudflare D1, Login, Benutzerrollen und echte Teamfähigkeit.
